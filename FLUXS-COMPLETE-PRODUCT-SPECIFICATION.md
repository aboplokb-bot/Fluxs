## 🎯 REAL-TIME FLUX DETECTION ALGORITHM

### **Input Variables (Real-Time):**
1. Sleep quality (last night): 1-5
2. Current mood: emoji (⚡😊😐🌫️😴)
3. Current energy: 1-5
4. Focus quality: High/Medium/Recovering/Low
5. Recent interruptions: Count
6. Task type attempted: Deep/Admin/Creative/Meeting
7. Time of day: User's historical peak/slump pattern
8. Caffeine/water intake: Optional input
9. Minutes since last break: Timer

### **Output:**
**Real-Time Flux Level (1.0-5.0) + Immediate Suggestion**

### **Example Calculation (Morning Peak):**

```
INPUT:
- Sleep: 4/5 = 0.8 points
- Mood: ⚡ Energized = 1.0 points
- Energy: 4/5 = 0.8 points
- Focus: High = 1.0 points
- Interruptions: 0 = 0 penalty
- Task: Deep work = match = 0 bonus
- Time: 9:30am (user's peak) = +0.3 bonus
- Last break: 2 hours ago = -0.1 (needs break soon)
────────────────────────────────
FLUX = (0.8 + 1.0 + 0.8 + 1.0 + 0 + 0 + 0.3 - 0.1) / 4.4 = 0.95 = 4.75/5

OUTPUT:
⚡ PEAK FLUX (4.75/5)
You're in the zone. Protect this.
Block 2 hours for your hardest work.
```

   - £99/month signals "serious tool" (not another freemium app)
   - Attracts quality users, filters out tire-kickers
